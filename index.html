githelper – github bookmarklet

Designed to add helpful UI elements to github.com

Features

  • Adds a link to your dashboard which toggles the Hashrocket projects under "watched repositories"
  • Adds a textbox to your root repository allowing you to live preview your readme file. (markdown supported, textile next)

Install

  1. Drag githelper to your bookmarks
  2. Visit your github dashboard
  3. Click the bookmarklet
